MyPlayer
===============

这是一个将指向音乐或者视频页面的链接转换为相应外站播放器的插件。目前支持虾米、优酷、土豆、爱奇艺、音悦台、乐视、56、哔哩哔哩、新浪的播放页面链接。另简单支持flash和html5音频视频。
用法如下（这里展示普通Markdown代码。html请自行脑补。）：

```
[龙凤恋曲](http://www.xiami.com/song/10907)
[视频: 一位老爸把他过去三个月每周六的早上拍下来后做了个合集](http://v.youku.com/v_show/id_XNjk1MTI5NTU2.html)
[欧派之歌·抖抖抖腿根本停不下来](http://www.bilibili.com/video/av1192762/)
```

插件通过js脚本检查指定范围内的链接，当匹配某个插件可识别的页面时，在前台转换为对应的官方外站播放器。

本插件具有“无侵入性”：由于文档中只是使用普通链接，在关闭、移除了插件后，文档内容不受影响，没有数据丢失或冗余，仅仅由点击链接展开播放器降级为打开播放页面，是一种可接受的降级方式。使用后不喜欢本插件的可以随意关闭并删除之。
